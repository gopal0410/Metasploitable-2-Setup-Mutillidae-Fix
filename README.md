## Project Overview

This project involves setting up the Metasploitable 2 vulnerable virtual machine and fixing a database-related error in the Mutillidae II web application. The goal of the project is to verify the proper functioning of the web application after resolving the database issue.

## Environment

*   Virtual Machine: Metasploitable 2

*   Host OS: Windows

*   Virtualization Platform: VMware

*   Network Mode: Bridged Adapter

*   Database: MySQL

## Project Tasks Performed

*   Imported and configured the Metasploitable 2 virtual machine.

*   Changed the network adapter to Bridged mode to enable network access.

## Verified that Metasploitable services and hosted websites were accessible via a browser.

*   Accessed the Mutillidae II web application and tested its functionality.

*   Encountered a database error while attempting to log in.

*   Identified and fixed the database configuration issue.

*   Re-tested Mutillidae II to confirm it was working correctly without errors.

## Verification

*   The following were verified as part of this project:

*   Metasploitable 2 running successfully

*   Network connectivity using bridged mode

*   Mutillidae II loading correctly in the browser

*   Database error resolved

*   Application functioning as intended after the fix
